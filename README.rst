Wordle Solver
=============

This program helps you solve `Wordle`_ puzzles.

.. _`Wordle`: https://www.powerlanguage.co.uk/wordle/


How to use
----------

Usage example:

.. code-block:: console

 $ ./solver.py -1 r -5 l -a udiceny -s _o_a_
 === 2 ===
 lobar
 molar
 polar
 solar
 volar
 $


Installation
------------

Install Python 3.8+, and run solver like so:

.. code-block:: console

   $ python3 solver.py
